# prompt-builder
Tool for creating optimized LLM prompts [Go To Prompt Builder Site](https://tolinsimpson.github.io/prompt-builder/)

Generative AI will give better output based on the keywords used. The prompt builder tool ensures the proper technical jargon and keywords are used for input prompts.

For example "Make me a concise optimal sandwich" may result in worse code output than "Create a small efficient sandwich". 
This tool outputs language LLMs will better understand.

There is also an included Claude skill for automatically optimizing input prompt language: [prompt-builder/SKILL.md](https://github.com/TolinSimpson/prompt-builder/blob/main/SKILL.md)
